# Fame Ticker

Real-time celebrity news and entertainment coverage. Breaking stories, red carpet fashion, music industry drama, and reality TV — delivered before it trends.

Built with Next.js 15, Tailwind CSS 3, and TypeScript. Deployed on Netlify.

**Production:** [fameticker.news](https://fameticker.news)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server with Turbopack |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npx tsc --noEmit` | Type check |

## Design

- **Brand:** Fame Ticker — gold (#C8A96E) on black (#0D0D0D)
- **Ticker:** Deep burgundy (#8B0000) with blush pink (#E8C4C4) Cormorant Garamond italic
- **Content:** Cream (#FDF5E6) backgrounds
- **Fonts:** Playfair Display (headlines), DM Sans (body), Bebas Neue (labels), Cormorant Garamond (ticker), Oswald (ticker label)

## Deploy

Push to `main` — Netlify auto-deploys.
